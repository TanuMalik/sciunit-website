Installation Guide

  1. Sciunit-CLI runs on most Linux distributions. Install the following sciunit dependencies with your distribution's package manager:

    • git
    • python-dev
    • python-setuptools
    • g++

  2. Download the Sciunit-CLI code from bitbucket with this link or with the following command:

    $ git clone -b Local https://bitbucket.org/geotrust/sciunit-cli.git

  3. Install Sciunit-CLI: 

    $ cd sciunit-cli/sciunit/client
    $ sudo python setup.py install

    And run the following commands: 

    $ cd ../..
    $ sudo python setup.py install globus
    $ sudo python setup.py install

    4. You are now ready to run the Sciunit-CLI! See the documentation page for details on using Sciunit-CLI
